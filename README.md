## Instalação

### Clone o projeto
```
git clone https://github.com/JoacyOliveira/backdoor.git
cd backdoor
```

### virtualenv
```
python -m venv venv
```
Windows
```
venv\Scripts\activate
```


## Instale as dependencias
```
pip install -r requirements.txt

```
